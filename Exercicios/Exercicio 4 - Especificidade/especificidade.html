Especificidade CSS

A especificidade no CSS determina quais regras de estilo são aplicadas a um elemento com base nos seletores, resolvendo conflitos entre regras.



Hierarquia de Especificidade


Inline Styles

Os estilos aplicados diretamente no atributo style de um elemento HTML têm a maior especificidade.

Exemplo: <div style="color: red;"></div>.


IDs

Seletores de ID têm uma alta especificidade.

Exemplo: #header { color: blue; }.


Classes, Pseudoclasse e Atributos

Seletores de classe, pseudoclasses e atributos têm uma especificidade média.

Exemplo: .menu { color: green; } ou a:hover { color: yellow; } ou [type="text"] { color: black; }.


Elementos e Pseudoelementos

Seletores de elementos (tags) e pseudoelementos têm a menor especificidade.

Exemplo: p { color: pink; } ou ::before { content: ""; }.



Regras de Desempate

Se duas regras tiverem a mesma especificidade, a última regra definida no CSS é aplicada. A ordem em que as regras aparecem no código também afeta qual estilo é aplicado.
