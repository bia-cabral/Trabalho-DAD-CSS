Box model

Conceito fundamental que defini posição dos elementos na tela, temos como atributos do box model:


O Content (conteúdo) é a área onde são exibidos o texto, as imagens e outros elementos. O Padding (preenchimento) é o espaço entre o conteúdo e a borda da caixa. A Border (borda) é a linha que envolve a caixa de conteúdo e o padding. Já a Margin (margem) é o espaço fora do elemento.


Oque fazer:

  1. Ajustar Margens: Mudar valores de margin-top e margin-left até que a princesa fique na mesma posição da princesa fantasma.

  2. Visualizar o Resultado: Ajuste conforme necessário até que a princesa esteja perfeitamente alinhada com a imagem fantasma.
Code

.pincesa-editor {

margin-left: px;

margin-top: px;

max-width: 8vw;

background-color: #1d1e22;

box-shadow: 0px 4px 30px rgba(0, 0, 0, 0.5);

border-radius: 8px;

}
