Position

A propriedade position no CSS é usada para definir como um elemento é posicionado na página. Existem várias opções que você pode usar:

  • static: Posição padrão, segue o fluxo normal do documento.

  • relative: O elemento é posicionado em relação ao seu elemento pai.

  • O elemento é posicionado em relação ao seu parente mais próximo com posição diferente de static.

  • O elemento é posicionado em relação à janela do navegador

  • O elemento alterna entre relative e fixed dependendo da rolagem.

Testando a Propriedade position

Neste exercício, você vai experimentar diferentes valores da propriedade position para mover um elemento na tela. Vamos usar dois elementos: uma princesa mãe e uma princesa filha.

Objetivo: Você deve experimentar diferentes valores da propriedade position até conseguir mover a princesa filha para o lado da princesa mãe.



Instruções:
  • No campo de entrada no codigo abaixo, digite um valor para a propriedade position (pode ser static, relative, absolute, fixed ou sticky) e clique no botão para aplicar a mudança.

  • Observe como a princesa filha se move na tela.

  • Continue experimentando até conseguir mover a princesa filha para o lado da princesa mãe.
Code

position: ;

max-width: 8vw;

background-color: #1d1e22;

box-shadow: 0px 4px 30px rgba(0, 0, 0, 0.5);

}
